ZooKeeper can be accessed via port {{ .Values.clientPort }} on the following DNS name from within your cluster:

    {{ template "zookeeper.client-service-name" . }}.svc.cluster.local


To connect to your ZooKeeper server run the following commands:

    export POD_NAME=$(kubectl get pods --namespace {{ .Release.Namespace }} -l "app={{ template "zookeeper.name" . }}" -o jsonpath="{.items[0].metadata.name}")
    kubectl exec -it $POD_NAME -- zkCli.sh
